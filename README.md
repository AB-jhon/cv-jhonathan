# 💼 CV Digital — Jhonathan Anota Badillo

CV interactivo desarrollado con React y Vite, desplegado en Vercel.

## 🚀 Demo en vivo
[Ver CV](https://cv-jhonathan-6118wl8qi-jhonathan-anotas-projects.vercel.app/)

## 🛠️ Tecnologías usadas
- React.js
- Vite
- JavaScript ES6+
- CSS-in-JS (estilos en línea)

## ✨ Características
- Modo oscuro / claro
- Diseño responsive
- Animaciones fade-in
- Línea de tiempo interactiva
- Tabs de experiencia laboral
- Foto de perfil

## 📦 Instalación local
```bash
git clone https://github.com/AB-jhon/cv-jhonathan
cd cv-jhonathan
npm install
npm run dev
```
